# University Hub — Frontend

React + Vite frontend for the University News Hub project.

## Stack
- **React 18** + **TypeScript** + **Vite**
- **TailwindCSS** + **shadcn/ui**
- Connects to the FastAPI backend (no Supabase)

## Setup

```bash
npm install
```

## Configuration

Create a `.env` file:
```
VITE_API_URL=http://localhost:8000/api/v1
```

## Run

```bash
npm run dev
# → http://localhost:5173
```
